MVP: Discord bot that provides gizz lyrics

- lyrics-service
  - app
    - main api
    - creates jobs in rabbitMQ queue
    - inserts artist/songs into posgres
    - fetch lyrics
  - lyrics-runner
    - recieves jobs from queue
    - fetches lyrics from genius (using library)
    - stores lyrics in sections
    - could move to multithreadeds

- discord-bot
  - discord integration (bot)
  - registers slash commands for app

- logger
  - default logger for services

- genius-library
  - provides access to Genius API/lyrics
